\section{Part A: Inference \& Uncertainty
quantification}\label{part-a-inference-uncertainty-quantification}

\subsection*{Task 1}\label{task-1}
\addcontentsline{toc}{subsection}{Task 1}

A single 1-hectare plot in a forest is surveyed and \(Y = 80\) birds are
counted. We can assume that the count
\(Y \sim \text{Poisson}(\lambda)\), where \(\lambda\) is the
\textbf{expected number of birds per hectare}, so that
\(\hat\lambda = Y = 80\) and \(\operatorname{Var}(Y) = \lambda\). Derive
a 95\% Wald confidence interval for \(\lambda\) and comment.

\subsection*{Task 2}\label{task-2}
\addcontentsline{toc}{subsection}{Task 2}

A great crested newt eDNA survey checks \(n = 20\) ponds and detects the
species at \(y = 3\) of them. Assuming \(Y \sim \text{Binomial}(n, p)\),
where \(p\) is the probability occurrence with a MLE estimate
\(\hat p = y/n = 0.15\).

\begin{enumerate}
\def\labelenumi{(\alph{enumi})}
\tightlist
\item
  Derive \(\text{Var}(\hat p)\) and compute a 95\% (Wald) confidence
  interval for \(\hat p\) (tip: recall that if
  \(X\sim\text{Bin}(n,p); \operatorname{Var}(X)=np(1-p)\).
\item
  what do you notice about this interval?
\end{enumerate}

\subsection*{Task 3}\label{task-3}
\addcontentsline{toc}{subsection}{Task 3}

Occupancy models are widely used in ecology to model species presence
while accounting for the fact that a species might present but
undetected during a survey. Let \(z\) denote whether a site is truly
occupied (\(z = 1\)) or not (\(z = 0\)), with \textbf{occupancy
probability} \(\Pr(z = 1) = \psi\). Each site is visited on multiple
occasions. On each visit we record a detection (\(y = 1\)) or
non-detection (\(y = 0\)), and we detect the species --- \emph{if it is
present} --- with \textbf{detection probability}
\(\Pr(y = 1 \mid z = 1) = p\). We assume that if the species is truly
absent we cannot detect it, \(\Pr(y = 1 \mid z = 0) = 0\).

A site is surveyed \(J\) independent times and the species is
\textbf{never detected}. A non-detection does not prove absence, so we
quantify our belief that the site is occupied with the posterior
probability \(\Pr(z = 1 \mid y_1 = 0, \dots, y_J = 0)\).

\begin{enumerate}
\def\labelenumi{(\alph{enumi})}
\tightlist
\item
  Write \(\Pr(\text{not detected in } J \text{ visits} \mid z = 1)\) and
  \(\Pr(\text{not detected in } J \text{ visits} \mid z = 0)\).
\item
  Use the \textbf{law of total probability} to write
  \(\Pr(\text{not detected})\) (i.e., the probability that species is
  never detected)
\item
  Use Bayes' theorem to obtain
  \(\Pr(z = 1 \mid y_1 = 0, \dots, y_J = 0)\).
\item
  A wetland is surveyed \(J = 2\) times for a species of frog that was
  not detected in neither of the surveys. Suppose that we know from
  previous studies that \(\psi = 0.8\) and \(p = 0.5\). Compute the
  posterior probability that frog is present at that site.
\end{enumerate}

\subsection*{Task 6}\label{task-6}
\addcontentsline{toc}{subsection}{Task 6}

\begin{enumerate}

steps:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Write the likelihood of the pilot study and show it has the form of a
  Beta distribution where the general Beta density
  \(\pi(p) \propto p^{a-1}(1-p)^{b-1}\).
\item
  Identify the likelihood and the prior for the main survey (recall that

  Derive the posterior distribution by multiplying the likelihood by the
  prior. What family results?
\item
  Compute the posterior mean of \(p\) using the fact that if
  \(X\sim\text{Beta}(a,b); \mathbb{E}(X) =\frac{a}{a+b}\))
\item
  Compute a 95\% credible intervals for \(p\) and comment (you can use
  \texttt{R} to compute the quantiles directly)
\end{enumerate}
